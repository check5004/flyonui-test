# FlyonUI 対話型セットアップガイド

Windows環境でNuxt.jsとUIフレームワーク「FlyonUI」を使用して、新しいフロントエンドプロジェクトを立ち上げるための対話型手順書です。

## 📋 概要

このプロジェクトは、開発者が効率的にFlyonUIベースのNuxt 3アプリケーションを構築できるよう設計された対話型ガイドです。ステップバイステップの手順に従うことで、短時間で本格的なフロントエンドプロジェクトの基盤を構築できます。

### 🎯 主な特徴

- **対話型UI**: アコーディオン形式で段階的に手順を表示
- **コピー機能**: コードブロックのワンクリックコピー機能
- **レスポンシブデザイン**: デスクトップ・モバイル両対応
- **日本語対応**: 完全日本語での説明とガイド
- **実践的**: 実際のプロジェクトで使用可能な設定

## 🛠️ 技術スタック

### フロントエンド
- **フレームワーク**: Nuxt 3
- **UIフレームワーク**: FlyonUI + Tailwind CSS
- **状態管理**: Pinia
- **国際化**: @nuxtjs/i18n
- **リンター**: ESLint (@nuxt/eslint-config)
- **アイコン**: @iconify/tailwind4

### 開発環境
- **OS**: Windows 10/11
- **Node.js**: LTS版 (v18.x, v20.x推奨)
- **パッケージマネージャー**: npm

## 🚀 クイックスタート

### 前提条件

以下の環境が整っていることを確認してください：

- ✅ Windows 10/11
- ✅ PowerShell または コマンドプロンプト
- ✅ Node.js LTS版がインストール済み
- ✅ インターネット接続

### 使用方法

1. **プロジェクトのクローンまたはダウンロード**
   ```bash
   git clone <repository-url>
   cd flyonui-setup-guide
   ```

2. **ブラウザで開く**
   - `index.html` をブラウザで開く
   - または、ローカルサーバーを起動：
   ```bash
   # Python 3の場合
   python -m http.server 8000
   
   # Node.jsの場合
   npx serve .
   ```

3. **ガイドに従ってセットアップ**
   - サイドバーのナビゲーションを使用
   - 各ステップのアコーディオンを展開
   - コードブロックの「Copy」ボタンでコマンドをコピー

## 🔧 カスタマイズ

### テーマの変更
`index.html` 内のCSS変数を変更することで、カラーテーマをカスタマイズできます：

```css
:root {
  --primary-color: #3730a3;
  --secondary-color: #e0e7ff;
  --text-color: #1f2937;
  --bg-color: #f9fafb;
}
```

### コンテンツの追加
新しいセクションを追加する場合は、以下の構造に従ってください：

```html
<section id="new-section" class="mb-12 scroll-mt-20">
  <h2 class="text-3xl font-bold border-b pb-2 mb-4">新しいセクション</h2>
  <!-- コンテンツ -->
</section>
```

## 🐛 トラブルシューティング

### よくある問題

**Q: コードブロックのコピーが動作しない**
A: ブラウザのセキュリティ設定を確認してください。HTTPS環境での実行を推奨します。

**Q: スタイルが正しく表示されない**
A: ブラウザのキャッシュをクリアしてページを再読み込みしてください。

**Q: モバイルで表示が崩れる**
A: ビューポートの設定を確認し、適切なメタタグが設定されているか確認してください。

問題が解決しない場合は、以下の情報と共にIssueを作成してください：

- 使用しているブラウザとバージョン
- エラーメッセージ（ある場合）
- 再現手順
- 期待される動作
