はじめに

このガイドは、Windows環境でNuxt.jsとUIフレームワーク「FlyonUI」を使用して、新しいフロントエンドプロジェクトを立ち上げるための対話型手順書です。FlyonUIはTailwind CSSと完全に統合されており、効率的な開発を支援します。

技術スタック

  • フレームワーク: Nuxt 3
  • UIフレームワーク: FlyonUI, Tailwind CSS
  • 状態管理: Pinia
  • 国際化: @nuxtjs/i18n
  • リンター: ESLint

前提条件

セットアップを開始する前に、お使いの環境が以下の要件を満たしていることを確認してください。

  • ✓
    OS: Windows 10 / 11
  • ✓
    ターミナル: PowerShell または コマンドプロンプト
  • ✓
    Node.js: LTS版 (例: v18.x, v20.x) がインストール済みであること。
    公式サイトからダウンロード →

プロジェクトセットアップ

ここからはターミナルを使い、Nuxtプロジェクトの雛形作成と依存関係のインストールを行います。各ステップのアコーディオンを開いて手順を確認してください。

作業用のディレクトリを作成し、移動します。以下のコマンドを順番に実行してください。

mkdir C:\dev
cd C:\dev
mkdir my-flyonui-project
cd my-flyonui-project

対話形式でNuxtプロジェクトを作成します。以下のコマンドを実行し、ガイドに従って選択してください。

npm create nuxt@latest

UI frameworkは `None`、Linting toolsは `ESLint` を選択するのがおすすめです。

FlyonUIや関連モジュールをインストールします。

ランタイム依存関係:

npm i pinia @pinia/nuxt @nuxtjs/i18n @headlessui/vue @heroicons/vue flyonui

開発時依存関係:

npm i -D @nuxt/eslint-config eslint @nuxtjs/tailwindcss @iconify/tailwind4

開発サーバーを起動し、初期ページが表示されるか確認します。

npm run dev

ブラウザで http://localhost:3000 を開き、Nuxtのウェルカムページが表示されたら成功です。確認後、ターミナルで `Ctrl + C` を押してサーバーを停止してください。

各種設定

インストールしたモジュールをNuxtプロジェクトで有効にするための設定ファイルを作成・編集します。

プロジェクトのルートにある `nuxt.config.ts` を以下の内容で更新します。

export default defineNuxtConfig({
  devtools: { enabled: true },
  modules: [
    '@pinia/nuxt',
    '@nuxtjs/i18n',
    '@nuxtjs/tailwindcss'
  ],
  i18n: {
    locales: ['ja', 'en'],
    defaultLocale: 'ja',
  },
  css: [
    '~/assets/css/tailwind.css'
  ],
})

1. `tailwind.config.js` を作成

プロジェクトのルートに以下の内容で `tailwind.config.js` を作成します。

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./components/**/*.{js,vue,ts}",
    "./layouts/**/*.vue",
    "./pages/**/*.vue",
    "./plugins/**/*.{js,ts}",
    "./app.vue",
    "./error.vue",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

2. `assets/css/tailwind.css` を作成

`assets/css/` ディレクトリに `tailwind.css` を作成し、FlyonUIをプラグインとして読み込みます。

@import "tailwindcss";

@plugin "flyonui";
@plugin "@iconify/tailwind4";

@import "flyonui/variants.css"; 
@source "./node_modules/flyonui/dist/index.js";

サンプルページ作成

アプリケーションの基本的な骨格となるファイルを作成し、FlyonUIのコンポーネントを試します。

以下のディレクトリがプロジェクトルートに存在することを確認してください（なければ作成します）。

/components
/components/ui
/layouts
/pages
/stores

全ページ共通のレイアウトを作成します。

<template>
  <div class="bg-gray-50 min-h-screen">
    <header class="bg-white shadow-sm">
      <nav class="container mx-auto px-4 py-3">
        <h1 class="text-xl font-bold">FlyonUI Project</h1>
      </nav>
    </header>

    <main class="container mx-auto px-4 py-8">
      <slot />
    </main>

    <footer class="text-center py-4 text-gray-500">
      <p>&copy; 2025 My FlyonUI Project</p>
    </footer>
  </div>
</template>

FlyonUIのクラスを使ったサンプルページを作成します。

<template>
  <div>
    <h2 class="text-2xl font-semibold mb-4">Welcome to FlyonUI!</h2>
    <p class="mb-6">This is a sample page using FlyonUI components with Nuxt 3.</p>

    <div class="flex items-center gap-4">
      <button class="btn btn-primary">Primary Button</button>
      <button class="btn btn-secondary">Secondary Button</button>
      <span class="badge badge-info">Info Badge</span>
    </div>
  </div>
</template>

`app.vue` を編集し、レイアウトとページが描画されるようにします。

<template>
  <div>
    <NuxtLayout>
      <NuxtPage />
    </NuxtLayout>
  </div>
</template>

最終確認

すべての設定が完了しました。再度、開発サーバーを起動して表示を確認しましょう。

npm run dev

ブラウザで http://localhost:3000 を開きます。作成したレイアウトの中に、FlyonUIのスタイルが適用されたボタンとバッジが表示されていれば、セットアップは成功です！
